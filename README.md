# File Description
## Petersburg.java
This file is primarily for the sake of making the frequency table of a billion game shown in the presentation. 
When working up to a billion, the program takes time so feel free to do any edits that you can that will maximize efficiency.
## PowerballSim.java
This file, as the name implies, is a Powerball Simulation.
It does not keep track of when winnings are won but just keeps track that they are being made.
This files takes the longest of all to complete, compared to the rest, so if any efficiencies can be made please make them. 
## PeterAvgMed.java
This file similates a set of games (in our case 20 games) then finds the average and median of said set of 1000 times.
The file then finds the overall average and the overall median for comparison.
This program is a precurssor to the PetersburgData.java where more information can be derived.
## PetersburgData.java

This allows for more data manipulation to be avaliable and seeing why things may happen.
